# Guías en Español

Este directorio contiene guías y documentación en español para el equipo.

## Propósito

- Guías de desarrollo en español
- Tutoriales específicos del proyecto
- Documentación de procesos internos
- Onboarding de nuevos desarrolladores

## Contenido Sugerido

- **onboarding.md**: Guía de inicio para nuevos desarrolladores
- **flujo-desarrollo.md**: Proceso de desarrollo del equipo
- **deployment.md**: Guía de deployment y releases
- **base-datos.md**: Estructura y convenciones de la base de datos
- **arquitectura.md**: Visión general de la arquitectura del sistema
- **troubleshooting.md**: Solución de problemas comunes

## Idioma

Toda la documentación en este directorio debe estar en español, usando "tú" (no "usted") para mantener un tono cercano y directo.
