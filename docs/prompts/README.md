# Prompts y Plantillas

Este directorio contiene prompts de IA, plantillas de código y recursos reutilizables.

## Propósito

- Prompts para Claude/ChatGPT relacionados con el proyecto
- Plantillas de código reutilizables
- Instrucciones estándar para generación de código
- Contexto del proyecto para asistentes de IA

## Contenido Sugerido

- **prompts-desarrollo.md**: Prompts para desarrollo de features
- **prompts-refactoring.md**: Prompts para refactorización
- **prompts-testing.md**: Prompts para generación de tests
- **contexto-proyecto.md**: Contexto general del proyecto para IA
- **plantillas/**: Plantillas de componentes, API endpoints, etc.

## Uso

Estos prompts ayudan a mantener consistencia al usar asistentes de IA para desarrollo, documentación y testing del proyecto.
