# Templates de Google Tag Manager (Archivo)

Este directorio contiene templates, configuraciones y documentación histórica de Google Tag Manager.

## Propósito

- Templates de GTM utilizados previamente
- Configuraciones de tags, triggers y variables
- Documentación de implementaciones pasadas
- Referencia para auditorías o migraciones

## Contenido

- Archivos JSON exportados de GTM
- Documentación de configuración de eventos
- Screenshots de configuraciones
- Notas sobre implementaciones específicas

## Nota

Este es un directorio de archivo. Para configuraciones activas de GTM, consulta la documentación actual o el panel de GTM en producción.
