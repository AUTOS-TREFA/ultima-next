# Scripts Antiguos (Archivo)

Este directorio contiene scripts deprecados, obsoletos o reemplazados por soluciones más modernas.

## Propósito

- Archivo de scripts que ya no se usan en el proyecto actual
- Referencia histórica de procesos automatizados anteriores
- Recuperación de lógica en caso necesario
- Documentación de evolución de herramientas del proyecto

## Contenido Típico

- Scripts de build antiguos
- Herramientas de deployment deprecadas
- Utilidades reemplazadas por paquetes npm
- Scripts de migración ya aplicados

## Nota Importante

Estos scripts NO deben ejecutarse sin revisar primero su compatibilidad con la versión actual del proyecto. Están aquí únicamente como referencia histórica.
