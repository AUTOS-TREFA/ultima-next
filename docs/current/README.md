# Documentación Actual

Este directorio contiene la documentación más reciente y relevante del proyecto.

## Propósito

- Documentación activa del proyecto
- Especificaciones técnicas actuales
- Notas de desarrollo en progreso
- Referencia rápida para el equipo

## Uso

Coloca aquí toda la documentación que esté relacionada con el estado actual del proyecto, features en desarrollo, decisiones técnicas recientes y cualquier documentación que el equipo necesite consultar frecuentemente.

## Organización

- Mantén los archivos actualizados
- Mueve documentación obsoleta a `/docs/archive/`
- Usa nombres descriptivos para los archivos
- Incluye fechas cuando sea relevante
